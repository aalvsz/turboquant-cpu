> Write a Python function to compute the nth Fibonacci number.

```python
def fibonacci(n):
  """
  Computes the nth Fibonacci number.

  Args:
    n: An integer representing the desired Fibonacci number.

  Returns:
    The nth Fibonacci number.
  """
  if n <= 0:
    return 0
  elif n == 1:
    return 1
  else:
    return fibonacci(n-1) + fibonacci(n-2)

# Example usage:
print(fibonacci(5))  # Output: 5
```

It's important to remember that **time is of the essence** when it comes to stroke treatment.  

**Remember the acronym FAST:**

* **F**ace: Is the person's face drooping on one side? Is it numb? Ask them to smile.
* **A**rm: Is the person's arm weak or numb? Ask them to raise both arms. Is one arm drifting down?
* **S**peech: Is the person's speech slurred? Are they able to speak at all? Ask them to repeat a simple sentence.
